## 1. 프로젝트 개요

**BioInvest Daily Digest**는 전 세계 의료, 제약, 바이오테크 분야의 투자 핵심 뉴스를 실시간으로 큐레이션하여 전문가 및 투자자들에게 제공하는 AI 기반 뉴스레터 플랫폼입니다. Google Gemini AI를 활용하여 방대한 글로벌 뉴스 중 시장 영향력이 큰 TOP 10 뉴스를 선별하고, 이를 한국어와 영어로 병기하여 제공합니다.

### 1.1 시스템 목적
* 글로벌 바이오/제약/의료 투자 정보를 AI가 자동 큐레이션
* 투자자 관점의 핵심 뉴스 TOP 10 일일 제공
* 한국어·영어 병기로 국내외 투자자 모두 활용 가능
* 이메일 뉴스레터, PDF/PPT 등 다양한 포맷으로 배포

### 1.2 주요 특징
| 항목 | 설명 |
|---|---|
| **AI 큐레이션** | Google Gemini AI가 임상 결과, FDA 승인, M&A 등 투자 중요도 기준으로 뉴스 자동 선별 |
| **다국어 지원** | 모든 뉴스를 한국어·영어 병기로 제공 (제목, 요약, 투자 포인트 포함) |
| **실시간 업데이트** | 버튼 클릭 한 번으로 최신 글로벌 뉴스 즉시 로드 |
| **멀티 채널 배포** | 이메일 발송, PDF 저장, PPT 생성, 텍스트 복사 등 다양한 배포 방식 지원 |
| **투자 심리 지수** | 각 뉴스별 시장 심리를 0~100 점수로 시각화 |

## 2. 핵심 기능

### 2.1 AI 뉴스 큐레이션 알고리즘
Google Gemini 3 Flash 모델을 활용한 자동 뉴스 선별 시스템입니다.

**큐레이션 프로세스:**
1. AI가 매일 실시간으로 글로벌 바이오/제약 뉴스를 검색
2. 투자 중요도 기준으로 가중치 부여 (임상 2/3상 결과 > FDA/EMA 승인 > M&A > 혁신기술)
3. 백엔드 서버에서 URL 실제 접속 가능 여부(200 OK) 실시간 검증
4. 유효하지 않은 URL 자동 제거
5. 제목, 요약, 투자 포인트를 한글/영어 병기로 생성

**주요 가중치 카테고리:**
* **높음 (High):** 임상 시험 결과(2/3상), FDA/EMA 승인 결정
* **중간 (Medium):** 대규모 M&A, 파트너십, 대규모 투자 유치
* **일반 (Normal):** 혁신 기술(유전자 편집, AI 신약 개발), 적응증 확대

### 2.2 이메일 뉴스레터 시스템
Resend API 기반의 프리미엄 HTML 이메일 뉴스레터 발송 시스템입니다.
* **발송 엔진:** Resend API (resend.com)
* **이메일 디자인:** Playfair Display 세리프 폰트 기반 프리미엄 레이아웃
* **반응형 지원:** 모바일 및 데스크톱 최적화 HTML 템플릿
* **다중 수신자:** 여러 수신자를 동시 관리하고 한 번의 클릭으로 전체 발송
* **콘텐츠 구성:** TOP 10 뉴스 전체 포함 (한/영 제목, 요약, 투자 포인트, 전문가 코멘트)

### 2.3 SNS 및 메모장 공유 (Copy All)
텍스트 기반의 시각적 디자인을 유지하면서 클립보드 복사를 지원합니다.
* 윈도우 메모장 등 구형 텍스트 에디터에서 한글이 깨지는 현상 방지
* 특수 유니코드 대신 표준 기호(`-`, `[]`)를 사용한 Notepad-Friendly 포맷
* UTF-8 인코딩 완벽 지원으로 복사-붙여넣기 시 가독성 확보
* 섹션별 구분선(`---`)을 통한 피드 가독성 확보

### 2.4 데이터 시각화
Recharts 라이브러리를 활용한 인터랙티브 차트 대시보드를 제공합니다.
* **카테고리 분포:** 도넛 차트 (뉴스 카테고리별 비중 분석)
* **투자 심리 지수:** 바 차트 (각 뉴스별 시장 심리 점수 시각화, 0~100점)
* **섹터 레이더:** 레이더 차트 (임상·규제·M&A·기술혁신 등 다차원 투자 강도)

### 2.5 다운로드 기능
* **PDF 다운로드:** 현재 대시보드를 `html2canvas`와 `jsPDF`를 통해 고해상도 PDF로 저장
* **PPT 다운로드:** `pptxgenjs` 라이브러리로 각 뉴스를 개별 슬라이드로 프로그래매틱 생성
* **텍스트 복사:** 전체 뉴스를 Notepad-Friendly 텍스트 포맷으로 클립보드 복사

## 3. 기술 스택

### 3.1 프론트엔드
* **React 18:** 컴포넌트 기반 UI 프레임워크, 최신 Hook API 활용
* **TypeScript:** 타입 안전성 보장, 런타임 오류 사전 방지
* **Tailwind CSS:** 유틸리티 퍼스트 CSS 프레임워크
* **Recharts:** SVG 기반 인터랙티브 차트 라이브러
* **Lucide Icons:** 일관된 아이콘 시스템
* **Framer Motion:** 카드 진입 애니메이션 및 호버 효과

### 3.2 백엔드 (Full-Stack 환경 시)
* **Node.js:** 서버 런타임 환경
* **Express:** 이메일 프록시 서버, Resend API와 통신
* **fetch API:** URL 유효성 검증 (200 OK 체크)

### 3.3 AI 및 외부 서비스
* **Google Gemini API:** Gemini 3 Flash/Pro 모델 (뉴스 큐레이션 및 한/영 번역)
* **Resend API:** 이메일 발송 서비스, 도메인 인증 지원
* **Google AI Studio:** 프리뷰 환경 및 API 키 관리

* 실시간 감성 지수 바 애니메이션

## 5. 설치 및 환경 설정

### 5.1 사전 요구사항
* Node.js 18.0 이상
* npm 또는 yarn 패키지 매니저
* Google Gemini API Key
* Resend API Key

### 5.2 패키지 설치
프로젝트 루트 디렉토리에서 다음 명령어를 실행합니다:
```bash
npm install
```

### 5.3 환경 변수 설정
프로젝트 루트에 `.env` 파일을 생성하고 아래 내용을 입력합니다.
```env
GEMINI_API_KEY=your_gemini_api_key_here
RESEND_API_KEY=your_resend_api_key_here
FROM_EMAIL=your_verified_domain_email_here
```

### 5.4 개발 서버 실행
```bash
npm run dev
```
브라우저에서 `http://localhost:3000` (또는 지정된 포트)로 접속합니다.

## 6. 사용 가이드

### 6.1 AI 뉴스 큐레이션
1. 좌측 사이드바의 **[Refresh News]** 버튼 클릭
2. 로딩 완료 후 TOP 10 뉴스가 대시보드에 표시됨

### 6.2 뉴스 카드 조작
* **카드 클릭:** 상세 내용(영문 요약, 투자 포인트, 원문 링크) 펼치기/접기
* **복사 버튼:** 개별 기사 우측 상단의 `[Copy]` 버튼으로 해당 기사만 클립보드 복사
* **원문 읽기:** 펼쳐진 카드 하단의 `Read Source` 링크로 원문 기사 이동

### 6.3 이메일 뉴스레터 발송
1. 좌측 사이드바에서 **[Send Email]** 클릭
2. 팝업 창에서 수신자 이메일 주소를 입력 후 `Enter` 키로 추가
3. 수신자 목록 확인 후 **[Send to N Recipients]** 버튼 클릭
4. 발송 성공 시 초록색 확인 메시지 표시 및 자동 닫힘

### 6.4 다운로드 및 공유
* **PDF 다운로드:** 사이드바의 `[Download PDF]` 클릭 시 대시보드 UI를 A4 사이즈 PDF로 변환
* **PPT 다운로드:** 사이드바의 `[Download PPT]` 클릭 시 슬라이드 형태의 프레젠테이션 파일 생성
* **전체 복사:** 사이드바의 `[Copy All (Notepad)]` 클릭 시 전체 텍스트 복사

## 7. 보안 주의사항

### 7.1 API 키 보안 수칙
* **필수:** `.env` 파일을 `.gitignore`에 반드시 추가하여 Git 저장소에 커밋되지 않도록 해야 합니다.
* API 키를 채팅, SNS, 이메일 등 외부에 절대 노출하지 않기
* 프론트엔드 소스코드에 API 키 직접 하드코딩 금지 (환경변수 활용)
* 주기적으로 API 키 재발급 및 교체

### 7.2 프로덕션 환경 권장사항
* 환경 변수는 서버 환경 변수 또는 시크릿 매니저로 관리
* HTTPS 적용으로 전송 구간 암호화
* Resend 발신 도메인 DNS 인증 완료 후 운영

## 9. 자주 묻는 질문 및 문제 해결 (FAQ)

**Q1. AI 새로고침이 작동하지 않습니다.**
* **원인:** Gemini API Key가 유효하지 않거나 한도를 초과함.
* **해결:** 환경변수(`.env`)에 키가 올바르게 설정되었는지 확인하고, Google AI Studio에서 할당량을 확인하세요.

**Q2. 이메일이 스팸함으로 분류됩니다.**
* **원인:** Resend 도메인 미인증 상태로 발송됨.
* **해결:** Resend 대시보드에서 도메인 DNS 레코드(SPF, DKIM, DMARC)를 추가하여 인증을 완료해야 합니다.

**Q3. PDF 다운로드 시 한글이 깨져서 표시됩니다.**
* **해결:** 현재 시스템은 `html2canvas`를 통해 이미지를 캡처하므로 한글이 깨지지 않습니다. 단, 향후 텍스트를 직접 주입(`pdf.text()`)할 경우를 대비해 `exportUtils.ts` 내에 한글 폰트 Base64 등록 로직이 준비되어 있습니다.

**Q4. 뉴스 URL이 작동하지 않습니다.**
* **원인:** 외부 뉴스 사이트의 URL 변경 또는 접근 제한(Paywall).
* **해결:** AI가 최대한 유효한 URL을 찾도록 프롬프트가 설정되어 있으나, 간혹 끊어진 링크가 발생할 수 있습니다. 기사 제목으로 직접 검색을 권장합니다.
